## 📝 Sobre o Projeto
O **TheBestOF-You** é uma plataforma web voltada para a saúde e bem-estar, com o objetivo de automatizar a criação de dietas e treinos personalizados. Utilizando inteligência artificial, o sistema gera planos adaptados às necessidades específicas de cada usuário.

## ✨ Funcionalidades
* **Autenticação Segura:** Cadastro e login de usuários.
* **IA Generativa:** Integração com a API do DeepSeek para geração automática de dietas e treinos.
* **Acompanhamento:** Ferramentas para monitorar a evolução física do usuário.
* **Customização:** Planos alimentares e rotinas de treino adaptáveis.

## 🛠️ Tecnologias Utilizadas
O projeto segue uma arquitetura Cliente-Servidor:
* **Frontend:** HTML5, CSS3 e JavaScript.
* **Backend:** PHP.
* **Banco de Dados:** MySQL.
* **IA:** API DeepSeek.
